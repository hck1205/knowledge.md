# CLO Virtual Fashion 주요 성과

## 재직 기간

2023년 10월 10일 클로버추얼패션에 입사해 현재까지 재직 중이다.

## 주요 성과

- CLO Virtual Fashion Closet 플랫폼의 PLM Plugin 프로젝트를 KR 오피스에서 Owner로 리딩했고, Global·India 조직(3개 오피스) 간 협업 구조와 개발 프로세스를 정립했다. India FE 팀의 아키텍처 방향성과 코드 품질 기준을 정의해 분산 조직의 개발 일관성과 생산성을 개선했다.

- Closet 서비스와 Plugin 간 데이터 및 인증 흐름을 분석했고, Integration Layer를 분리하는 구조로 리팩토링했다. 주요 모듈 간 결합도를 낮추고 기능 확장 시 영향 범위를 기존 대비 약 30% 이상 축소했으며, Cross-office 커뮤니케이션 이슈를 줄였다.

- SSR 환경에서 발생하던 중복 API 호출을 제거했고, CSR 전환 및 React Query 캐싱과 HTTP Cache 전략을 적용했다. 그 결과 API 호출을 약 40% 줄이고 서버 부하를 약 30% 절감했으며, Core Web Vitals를 개선했다. LCP는 3.2초에서 1.8초로 약 44% 개선했다.

- 대용량 데이터(수천~수만 건) 렌더링 환경에서 Virtual Scroll, Web Worker, Optimistic UI 기반 비동기 처리 구조를 도입했다. UI thread blocking을 최소화했고, 체감 응답 속도를 약 50% 개선했으며, Lighthouse Performance Score를 65에서 92로 향상했다.

- 3D Viewer와 2D Canvas 기반 Annotation 시스템에서 좌표계, 히트테스트, 렌더링 구조를 설계하고 구현했다. 복잡한 그래픽 인터랙션 환경에서도 안정적인 성능을 확보했다.

- Data Schema와 Design System을 기반으로 데이터 구조와 UI Layer를 표준화했다. 공통 컴포넌트 재사용성을 높였고, 프론트엔드에서 독립적인 개발이 가능하도록 구조를 개선해 기능 개발 리드타임을 약 20~30% 단축했다.

- 웹 기반 문서/데이터 편집 플랫폼에서 기술 스택 선정과 구조 설계에 참여했다. ideation 단계부터 기획 및 UX 설계 과정에 기여했고, 주요 기능 정의와 개발을 리딩하며 초기 제품 방향성 수립에 기여했다.
