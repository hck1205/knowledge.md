# CLO Virtual Fashion 재직 개요

## 기본 정보

- 회사명: CLO Virtual Fashion
- 입사일: 2023-10-10
- 재직 상태: 현재 재직 중

## 한 줄 요약

CLO Virtual Fashion의 Closet 플랫폼과 PLM Plugin 관련 프로젝트를 프론트엔드 중심으로 리딩하며, 제품 구조 개선, 성능 최적화, 협업 프로세스 정립, 초기 제품 방향성 수립에 기여했다.

## 주요 맥락

- KR 오피스에서 PLM Plugin 프로젝트를 Owner로 리딩했다.
- Global 및 India 조직과 협업하는 구조를 정립했다.
- 분산 조직 환경에서 개발 일관성과 생산성을 높이는 방향으로 구조와 프로세스를 개선했다.
