# 제품 정의

## 핵심 컨셉

다들픽은 두 가지 선택지를 보여주고, 사용자가 하나를 선택해 투표하면 전체 결과를 확인할 수 있는 간단한 앱이다.

## 토픽 특징

토픽은 누구나 한 번쯤 생각해볼 만한, 살짝 논쟁적이거나 취향이 갈리는 주제를 중심으로 가져간다.

예시:

- 짜장면 vs 짬뽕
- 톰 크루즈 vs 브래드 피트
- 이란 vs 이스라엘

## 사용자 경험 흐름

1. 사용자가 앱에 들어온다
2. 하나의 토픽과 두 개의 선택지를 본다
3. 둘 중 하나를 선택해 투표한다
4. 전체 투표 결과를 확인한다
5. 다음 토픽으로 넘어간다

## 제품의 매력 포인트

- 진입 장벽이 낮다
- 생각할 거리가 있어서 눌러보게 된다
- 결과를 바로 볼 수 있어 반응성이 좋다
- 한 번 시작하면 여러 토픽을 연속으로 보게 만들 수 있다

## 초반 제품 원칙

- 한 화면에서 이해 가능한 구조를 유지한다
- 불필요한 기능은 넣지 않는다
- 토픽 품질이 제품 재미를 좌우하므로, 주제 선정 기준이 중요하다

## 토픽 운영 메모

- 취향, 문화, 인물, 라이프스타일, 사회 이슈 등 여러 축에서 주제를 모을 수 있다
- 너무 공격적이거나 위험한 표현은 피하고, 앱이 소모적인 싸움판처럼 보이지 않게 조절할 필요가 있다
- 초반에는 수동 큐레이션 방식으로 토픽을 모으는 편이 안정적일 수 있다
