# everyonePick / 다들픽

## 한눈에 보기

- 프로젝트명: everyonePick
- 서비스명: 다들픽
- 형태: 회사 동료와 함께 진행하는 사이드 프로젝트
- 팀 구성: 나(모바일 앱 / Android), emmet-신영민(서버)
- 핵심 아이디어: 누구나 한 번쯤 고민해볼 만한 논쟁형 주제를 두 가지 선택지로 제시하고, 한 가지에 투표한 뒤 결과를 확인하는 아주 간단한 앱

## 현재 상태

- 모바일 앱은 Android 우선으로 개발 시작
- 서버는 emmet이 담당
- 서버 쪽은 현재 push 서버 개발 완료
- 다음 서버 작업은 login 기능 개발 예정
- 업무 트래킹과 협업은 ClickUp으로 진행
