# 프로젝트 개요

## 프로젝트 배경

회사 동료 emmet-신영민씨와 함께 가볍게 시작해보는 사이드 프로젝트다.  
앱 이름은 `다들픽`, 프로젝트 폴더명은 `everyonePick`으로 정리한다.

## 프로젝트 한 줄 설명

서로 다른 두 선택지 중 하나를 고르고, 다른 사람들의 투표 결과도 바로 확인할 수 있는 간단한 모바일 투표 앱.

## 왜 이 프로젝트를 하는가

- 구현 범위가 작아서 빠르게 출시 경험을 쌓기 좋다
- 모바일과 서버를 분리해서 역할 분담하기 좋다
- 주제만 잘 잡으면 사용자 반응을 바로 확인할 수 있다
- 작은 제품으로도 기획, 개발, 협업, 운영 흐름을 연습할 수 있다

## 팀 구성

- 나: 모바일 앱 개발 담당
- emmet-신영민: 서버 개발 담당

## 개발 방향

- 초기에는 Android 앱만 개발
- 서버는 필요한 최소 기능부터 순차적으로 확장
- 복잡한 커뮤니티 기능보다 투표 경험 자체에 집중

## 프로젝트 원칙

- 기능은 최대한 단순하게 유지한다
- 첫 버전은 빠르게 만들고 빠르게 확인한다
- 논쟁형 토픽이지만 앱 사용성은 가볍고 직관적으로 만든다
- 역할이 나뉘어 있는 만큼 문서와 ClickUp으로 합의를 남긴다
