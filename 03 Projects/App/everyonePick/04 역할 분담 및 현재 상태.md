# 역할 분담 및 현재 상태

## 역할 분담

### 나

- 모바일 앱 개발 담당
- 우선 Android 앱 개발 진행

### emmet-신영민

- 서버 개발 담당

## 현재 진행 상태

### 모바일

- Android 앱 개발 예정
- 아직 초기 기획과 범위 정리가 중요한 단계

### 서버

- push 서버 개발 완료
- 다음 작업은 login 기능 개발 예정

## 현재 기준 협업 포인트

- 로그인 방식이 어떻게 정해지는지에 따라 모바일 구현 방향이 달라질 수 있다
- 모바일에서 필요한 API 목록을 먼저 정리해두면 서버와 맞추기 좋다
- push 서버를 어떤 시점에 앱에서 활용할지는 MVP 범위와 함께 다시 판단하면 된다

## 확인이 필요한 항목

- 로그인 방식: 자체 로그인인지, 소셜 로그인인지
- 사용자 식별 기준: 익명 허용 여부, 계정 필수 여부
- 투표 결과 조회 정책: 투표 전 결과 비공개인지, 투표 후 공개인지
- 토픽 제공 방식: 서버 랜덤 제공인지, 정렬 기준이 있는지
